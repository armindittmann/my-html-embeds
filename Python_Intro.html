Schritt 1: Was ist Programmieren & Python?

❯

Was ist Programmieren?

Stell dir vor, du gibst einem Computer eine detaillierte Anleitung, um eine bestimmte Aufgabe zu erledigen. Genau das ist Programmieren! Du schreibst diese Anleitungen (den sogenannten "Code") in einer speziellen Sprache, die der Computer verstehen kann.

Was ist Python?

Python ist eine solche Programmiersprache. Sie wurde von Guido van Rossum entwickelt und 1991 veröffentlicht. Python zeichnet sich durch eine klare und gut lesbare Syntax (die "Grammatik" der Sprache) aus, was sie besonders für Anfänger geeignet macht.

Warum Python lernen?

  • Anfängerfreundlich: Die Syntax ist relativ einfach zu verstehen.
  • Vielseitig: Webentwicklung, Datenanalyse, künstliche Intelligenz, Spieleentwicklung, Automatisierung – Python kann fast alles!
  • Große Community: Viele Entwickler nutzen Python, es gibt unzählige Hilfsquellen und fertige Code-Bausteine (Bibliotheken).
  • Hohe Nachfrage: Python-Kenntnisse sind auf dem Arbeitsmarkt sehr gefragt.

Schritt 2: Dein erstes Python-Programm

❯

Traditionell beginnt man mit einem Programm, das "Hallo, Welt!" ausgibt. Öffne einen Texteditor (nicht Word, sondern z.B. Notepad unter Windows, TextEdit unter macOS oder einen Code-Editor wie VS Code) und tippe Folgendes ein:

print("Hallo, Welt!")

Speichere die Datei unter einem Namen wie hallo.py (die Endung .py ist wichtig!).

Öffne nun die Kommandozeile (Terminal oder Eingabeaufforderung) auf deinem Computer, navigiere in den Ordner, in dem du die Datei gespeichert hast, und führe sie mit dem Befehl aus:

python hallo.py

Wenn alles geklappt hat, sollte der Text Hallo, Welt! auf dem Bildschirm erscheinen. Herzlichen Glückwunsch, du hast dein erstes Python-Programm geschrieben und ausgeführt!

Was ist hier passiert?

  • print() ist eine eingebaute Funktion in Python. Funktionen sind wiederverwendbare Code-Blöcke, die eine bestimmte Aufgabe erledigen – in diesem Fall etwas auf dem Bildschirm ausgeben.
  • "Hallo, Welt!" ist ein String, also eine Zeichenkette (Text). Strings werden in Python in Anführungszeichen (" oder ') gesetzt.

Kommentare

Du kannst Notizen in deinem Code hinterlassen, die Python ignoriert. Diese nennt man Kommentare. Sie beginnen mit einem #:

# Dies ist ein Kommentar und wird ignoriert
print("Dieser Text wird ausgegeben.") # Auch hier kann ein Kommentar stehen

Kommentare sind wichtig, um deinen Code für dich und andere verständlich zu machen.

Schritt 3: Variablen und Datentypen

❯

Variablen: Container für Daten

Variablen sind wie beschriftete Boxen, in denen du Werte speichern kannst, um sie später wiederzuverwenden. Du erstellst eine Variable, indem du ihr einen Namen gibst und ihr mit dem Gleichheitszeichen (=) einen Wert zuweist:

name = "Alice"
alter = 30
groesse_in_m = 1.75
ist_student = False

# Variablenwerte ausgeben
print(name)
print(alter)

Hier haben wir vier Variablen erstellt: name, alter, groesse_in_m und ist_student.

Wichtige Datentypen

Jeder Wert in Python hat einen bestimmten Typ. Die wichtigsten für den Anfang sind:

  • String (str): Text, in Anführungszeichen. Beispiel: "Hallo", 'Python'.
  • Integer (int): Ganze Zahlen. Beispiel: 10, -5, 0.
  • Float (float): Gleitkommazahlen (Zahlen mit Nachkommastellen). Beispiel: 3.14, -0.5. Wichtig: Dezimaltrennzeichen ist der Punkt (.)!
  • Boolean (bool): Wahrheitswerte, können nur True (wahr) oder False (falsch) sein. Nützlich für Entscheidungen im Code.

Python erkennt den Typ meist automatisch, wenn du einer Variablen einen Wert zuweist. Du kannst den Typ einer Variablen mit der Funktion type() überprüfen:

print(type(name))     # Gibt  aus
print(type(alter))    # Gibt  aus
print(type(groesse_in_m)) # Gibt  aus
print(type(ist_student))# Gibt  aus

Schritt 4: Operatoren

❯

Operatoren sind spezielle Symbole, die Operationen (wie Rechnen oder Vergleichen) durchführen.

Arithmetische Operatoren

Zum Rechnen mit Zahlen:

  • + : Addition (5 + 3 ergibt 8)
  • - : Subtraktion (5 - 3 ergibt 2)
  • * : Multiplikation (5 * 3 ergibt 15)
  • / : Division (5 / 3 ergibt 1.666...)
  • // : Ganzzahl-Division (5 // 3 ergibt 1, Rest wird verworfen)
  • % : Modulo (Rest der Division, 5 % 3 ergibt 2)
  • ** : Potenzierung (5 ** 3 ergibt 125, also 5 hoch 3)
zahl1 = 10
zahl2 = 3
print(zahl1 + zahl2) # Ausgabe: 13
print(zahl1 / zahl2) # Ausgabe: 3.333...

Vergleichsoperatoren

Zum Vergleichen von Werten. Das Ergebnis ist immer ein Boolean (True oder False).

  • == : Gleich (5 == 3 ist False)
  • != : Ungleich (5 != 3 ist True)
  • > : Größer als (5 > 3 ist True)
  • < : Kleiner als (5 < 3 ist False)
  • >= : Größer oder gleich (5 >= 5 ist True)
  • <= : Kleiner oder gleich (5 <= 3 ist False)
alter = 20
print(alter >= 18) # Ausgabe: True

Logische Operatoren

Zum Verknüpfen von Boolean-Werten:

  • and : Logisches UND (True and False ist False) - Beide Seiten müssen wahr sein.
  • or : Logisches ODER (True or False ist True) - Mindestens eine Seite muss wahr sein.
  • not : Logisches NICHT (not True ist False) - Kehrt den Wahrheitswert um.
hat_fuehrerschein = True
ist_volljaehrig = True
darf_fahren = hat_fuehrerschein and ist_volljaehrig
print(darf_fahren) # Ausgabe: True

Schritt 5: Entscheidungen treffen (if/else)

❯

Oft soll Code nur unter bestimmten Bedingungen ausgeführt werden. Dafür gibt es die if-Anweisung.

alter = 17

if alter >= 18:
    print("Du bist volljährig.")
    print("Du darfst wählen.")
else:
    print("Du bist noch nicht volljährig.")

print("Dieser Text wird immer ausgegeben.")

Wie funktioniert das?

  • Nach if steht eine Bedingung (etwas, das True oder False ergibt), gefolgt von einem Doppelpunkt (:).
  • Der Code-Block, der ausgeführt werden soll, WENN die Bedingung True ist, wird eingerückt (meist 4 Leerzeichen). Die Einrückung ist in Python sehr wichtig!
  • Optional kann ein else: folgen. Der Code-Block darunter wird ausgeführt, WENN die if-Bedingung False ist.
  • Mit elif (kurz für "else if") kannst du weitere Bedingungen prüfen:
note = 2

if note == 1:
    print("Sehr gut!")
elif note == 2:
    print("Gut.")
elif note == 3:
    print("Befriedigend.")
elif note == 4:
    print("Ausreichend.")
else:
    print("Nicht bestanden.")

Schritt 6: Wiederholungen (Schleifen)

❯

Schleifen erlauben es, Code-Blöcke mehrmals auszuführen.

Die for-Schleife

Ideal, wenn man weiß, wie oft etwas wiederholt werden soll, oder wenn man über Elemente einer Sequenz (z.B. Zahlen) gehen will.

# Zähle von 0 bis 4
print("Zählen von 0 bis 4:")
for i in range(5): # range(5) erzeugt die Zahlen 0, 1, 2, 3, 4
    print(i)

print("\nQuadratzahlen:")
# Gib die Quadratzahlen von 1 bis 5 aus
for zahl in range(1, 6): # range(1, 6) erzeugt 1, 2, 3, 4, 5
    print(f"{zahl} im Quadrat ist {zahl * zahl}") # f-String für formatierte Ausgabe

Die Variable i (oder zahl im zweiten Beispiel) nimmt nacheinander jeden Wert aus der Sequenz (hier von range() erzeugt) an. Der eingerückte Block wird für jeden dieser Werte ausgeführt.

Die while-Schleife

Wird ausgeführt, solange eine Bedingung True ist. Vorsicht: Man muss sicherstellen, dass die Bedingung irgendwann False wird, sonst entsteht eine Endlosschleife!

# Countdown von 5 bis 1
print("\nCountdown:")
counter = 5
while counter > 0:
    print(counter)
    counter = counter - 1 # Wichtig: Zähler verändern!

print("Start!")

Schritt 7: Eigene Werkzeuge bauen (Funktionen)

❯

Funktionen sind benannte Code-Blöcke, die du definieren und dann immer wieder aufrufen kannst. Das macht deinen Code organisierter und wiederverwendbar.

Du kennst bereits die eingebaute Funktion print(). Jetzt erstellen wir unsere eigene.

Funktion definieren

Man verwendet das Schlüsselwort def, gefolgt vom Funktionsnamen, Klammern () und einem Doppelpunkt :. Der Code der Funktion wird eingerückt.

# Definiert eine einfache Funktion
def begruessung():
    print("Hallo!")
    print("Willkommen zum Python-Kurs.")

# Definiert eine Funktion mit einem Parameter (Eingabewert)
def persoenliche_begruessung(name):
    print(f"Hallo, {name}!") # f-String nutzt den Wert des Parameters

# Definiert eine Funktion, die einen Wert zurückgibt
def addiere(zahl1, zahl2):
    summe = zahl1 + zahl2
    return summe # Gibt das Ergebnis zurück

Funktion aufrufen

Um eine Funktion auszuführen, schreibst du ihren Namen gefolgt von Klammern. Wenn die Funktion Parameter erwartet, übergibst du die Werte in den Klammern.

# Rufe die Funktionen auf
begruessung()

print("-" * 10) # Trennlinie

persoenliche_begruessung("Max")
persoenliche_begruessung("Anna")

print("-" * 10)

ergebnis = addiere(5, 3) # Speichere den Rückgabewert in einer Variable
print(f"5 + 3 = {ergebnis}")

print(f"10 + 20 = {addiere(10, 20)}") # Direkte Verwendung des Rückgabewerts

Funktionen helfen enorm dabei, komplexe Probleme in kleinere, handhabbare Teile zu zerlegen.
